C Basic Programming
This repository contains various exercises written in C to practice fundamental programming concepts.

Included Exercises:
Basic switch use: in making of function and execution

Math problems: find Area of different shapes,solving diffrent pattern series of numbers like factorial.

checker of alphabets: whether input is upper or not.

Prime Numbers: Find primes within a user-defined range.

Shape Printer: Generate patterns using characters.

Array Manipulation: Reversing arrays and finding the maximum element.

Pointer Usage: Basic pointer demonstrations.
